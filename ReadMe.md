# このソフトウェアについて #

Draw201608111739は私個人が学習目的で作成したソフトウェアです。

GDI, GDI+, DirectX9, の描画Frameworkをつくってみた。
`～Drawer`クラスを切替できる予定だが、今はまだできない。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express

## ライブラリ ##

* GDI+
* DirectX9
